How to use OnsenUI2.

preview

You should use $monaca preview in current directry.

This function use webpack-dev-server. It also prevents webpack from emitting the resulting files to disk. Instead it keeps and serves the resulting files from memory.

$monaca preview use setting of webpack.dev.config.js in current directry.

build

If you want use static file, You shold use $monaca transpile in current directry. Monaca CLI transpile static file in folder www/

$monaca preview use setting of webpack.prod.config.js in current directry.

OnsenUI2の使い方

プレビュー

$monaca previewを使えばリアルタイムにコードの変更をブラウザでプレビューすることができます。

この機能はwebpack-dev-serverを使って実現しており、 メモリ上でエミュレーションして変更を表示しているので、コンパイルして実在ファイルを生成しているわけではないことにお気をつけ下さい。

$monaca previewは、インストールフォルダにあるwebpack.dev.config.jsの設定を利用します.

ファイル構築

プレビューではなく、実在ファイルを生成するなら、$monaca transpileというコマンドをインストールフォルダで打つ必要があります。 すると、Monaca CLIはwww/以下に実在ファイルを生成します。

$monaca previewは、インストールフォルダにあるwebpack.prod.config.jsを利用します。
